| Feature | Details |

| **Freya** | 12x12 companion sprite (fly1, fly2, beam) ‚Äî purple/cyan quantum palette, floats alongside the player |

### AI Training API

```javascript
// Game control
window.numsyAI.getState()  // {alive, score, speed, freyaPower, freyaBeamActive, distanceMeters, ...}
window.numsyAI.act('jump') // 'jump', 'duck', or 'none'
window.numsyAI.reset()     // restart the game
window.numsyAI.onFrame(cb) // register per-frame callback for RL training loops
```

### FreyaUnits Conversion API

```javascript
// Unit conversion from browser console
window.FreyaUnits.convert(1, 'mi', 'km')      // 1.60934
window.FreyaUnits.convert(1, 'AU', 'ly')       // 1.581e-5
window.FreyaUnits.toMeters(100, 'nm')           // 1e-7
window.FreyaUnits.fromMeters(0.0254, 'mil')     // 1000
window.FreyaUnits.bestUnit(42195)               // {sym:'km', name:'Kilometer', m:1000}
window.FreyaUnits.listUnits()                   // all 27 units with meter values
```

## hexcast ‚Äî Live Video Hex Broadcast

Real-time video capture ‚Üí hex stream encoding ‚Üí cross-device broadcast with full latency benchmarking. Streams camera, screen, or test patterns through the same thermal/fax hex grid used in the notepad ‚Äî measuring encode, decode, round-trip, and jitter on every frame.

**[Launch hexcast](https://fornevercollective.github.io/uvspeed/web/hexcast.html)** | Also accessible from the notepad footer (`üì° hexcast` button)

| Feature | Details |
|---|---|
| üì∑ Camera Capture | Live webcam feed ‚Üí hex stream encoding at configurable resolution |
| üñµ Screen Capture | Screen share ‚Üí hex stream for remote display testing |
| ‚ñ¶ Test Pattern | Synthetic SMPTE-style color bars with moving scan line |
| üî• Encode Modes | Color Thermal, Grayscale, Fax B/W, Signal (green phosphor) |
| üìä Latency Chart | Real-time line graph: encode (orange), decode (purple), round-trip (green), jitter (yellow) |
| üì° BroadcastChannel | Cross-tab broadcast ‚Äî one tab sends, another receives |
| ‚è± Benchmark | Automated encode/decode speed test across all resolutions √ó modes |
| üì∏ Snapshot | Export current hex frame as PNG |
| ‚ö° Code Cell | Full JS scripting ‚Äî `hexcast.startCamera()`, `hexcast.benchmark()` |
| ‚ò¢ Terminal | CLI commands ‚Äî `camera`, `screen`, `bench`, `latency`, `export` |

### hexcast API

```javascript
window.hexcast.state              // {source, fps, latency, throughput, frameSize, ...}
window.hexcast.startCamera()      // open webcam ‚Üí hex stream
window.hexcast.startScreen()      // screen share ‚Üí hex stream
window.hexcast.startTestPattern() // synthetic color bars
window.hexcast.stopSource()       // stop capture
window.hexcast.startBroadcast()   // send frames via BroadcastChannel
window.hexcast.startReceive()     // receive frames from another tab
window.hexcast.benchmark()        // [{resolution, mode, encodeMs, throughputMB}, ...]
window.hexcast.snapshot()         // save hex frame as PNG
window.hexcast.setEncode('fax')   // switch encode mode
window.hexcast.setResolution(108) // set grid resolution
window.hexcast.setFPS(30)         // set target framerate
window.hexcast.exportJSON()       // full state + latency history
```
